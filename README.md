# LTFS_FinHack2


Holidays are taken from the [website](https://www.india.gov.in/calendar)

L&T Financial Services & Analytics Vidhya presents ‘Data Science FinHack 2’.
Amazing opportunity for all creative nerds to apply their data science & machine learning skillset to best solve a real business problem.
In this FinHack, you will develop a model for our interesting business challenge ‘Business Forecasting’.

To run the code:
* update the path of train and test files in ```config.py```
* update the path where to store the submission file in ```config.py```

Install all the requirments using ```pip install -r requirements.txt```
To run the main file ``` python main.py ```
